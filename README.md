# AJMERA-SALES-DASHBOARD
1) Short Description

The Ajmera Ecommerce Sales Dashboard is an interactive Power BI report designed to analyze overall ecommerce performance across states, categories, customers, and payment modes. The dashboard provides a consolidated view of sales amount, profit, quantity sold, and average order value (AOV), enabling data-driven business decisions through visual insights and quarterly filtering.

2) Tech Stack

Power BI Desktop – Dashboard development & reporting

DAX (Data Analysis Expressions) – KPI calculations (Amount, Profit, Quantity, AOV)

Power Query – Data cleaning, transformation, and modeling

Microsoft Excel / CSV – Sales and transaction data source

Data Modeling – Relationships between sales, customers, products, and states

3) Data Source

Ecommerce sales transaction data (Excel/CSV format)

Customer master data

Product category & sub-category data

State-wise sales data

Payment mode details (COD, UPI, Cards, EMI)

4) Features & Highlights

Key KPIs at a glance:

Total Sales Amount (87K)

Total Profit (882)

Total Quantity Sold (1181)

Average Order Value (24K)

Quarter-wise analysis using interactive slicers (Qtr 1–Qtr 4)

State-wise sales performance to identify top and low-performing regions

Category-wise quantity distribution (Clothing, Electronics, Furniture)

Monthly profit trend analysis (April–June)

Customer-wise sales contribution for identifying high-value customers

Payment mode analysis (COD, UPI, Debit Card, Credit Card, EMI)

Sub-category profit comparison highlighting top-performing products

Clean, dark-themed UI with intuitive visuals and cross-filtering
